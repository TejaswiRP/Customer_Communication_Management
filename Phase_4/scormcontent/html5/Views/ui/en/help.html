HELP
This screen describes how to use various buttons available to you throughout the module.
Menu: Allows for navigation to a specific page within the module.
Next: Allows for navigation to the next screen. The 'Next' button will remain inactive until the audio is complete and all the buttons are clicked on the screen.
Audio On/Off: Allows you to turn on/off the audio.
Page Counter: Displays the current screen number and the number of screens remaining in the course.
Transcript: Displays the audio script.
Back: Allows for navigation to the previous screen.
Resources: Allows you to view the resources available in this course.
Play/Pause: Select this button to play \or pause the screen animation as well as audio.
Replay: Select this button to replay a screen from the start.
